# RoadTribe

The Home of Roculture - A mobile app for road trip enthusiasts.

## About

RoadTribe is a social platform for road trip lovers to plan, track, and share their journeys. Connect with your tribe, join convoys, and explore the open road together.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Mobile Development

```bash
# Build and sync to iOS
npm run build
npx cap sync

# Open in Xcode
npx cap open ios

# Run on simulator
npx cap run ios
```
